# 函数得分

单个函数的副作用负担分。

%% [函数得分] = <加权语句数> * <传播能力数量> %%

- 【加权语句数】：函数体复杂度
- 传播能力数量：effectiveCaps 中**传播能力**（IO, Impure, Fallible, Async, Mutable）的个数。阻断能力（HandleFallible, HandleAsync, HandleMutable）不计入——它们不产生负担。
- 【未声明函数】按 5 个传播能力计算（最大惩罚）
- 纯函数（已声明且零传播能力）得分为 0，即使声明了阻断能力也是 0

结果四舍五入到小数点后一位。

所有函数按得分降序排列，取前 10 构成 topFunctions。
