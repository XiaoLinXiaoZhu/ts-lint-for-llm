# 参数可变性检测

判断函数参数是否为"非 readonly 引用类型"，用于【Mutable】的【自动检测】。

## 判定为**不可变**（不触发 Mutable）的情况

1. 值类型：string, number, boolean, undefined, null, void, enum 及对应字面量类型
2. 联合类型中所有引用类型成员都被 `Readonly<>` 包装
3. 纯函数签名类型（有 call signature 但无 property）
4. 类型节点文本以以下前缀开头：`readonly `, `Readonly<`, `ReadonlyArray<`, `ReadonlyMap<`, `ReadonlySet<`
5. 消费型接口：`Iterable<`, `Iterator<`, `IterableIterator<`, `Generator<`（含 Async 变体）, `ReadableStream`

## 判定为**可变**（触发 Mutable）的情况

参数类型是引用类型（object, array, interface, intersection），且不满足上述任何不可变条件。

## 示例

| 参数声明 | 判定 |
|----------|------|
| `state: State`（State 是 interface） | 可变 |
| `state: Readonly<State>` | 不可变 |
| `items: string[]` | 可变 |
| `items: readonly string[]` | 不可变 |
| `a: number, b: number` | 不可变 |
| `fn: (x: number) => void` | 不可变 |
