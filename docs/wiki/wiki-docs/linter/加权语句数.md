# 加权语句数

衡量函数体复杂度的指标，用于计算【函数得分】。

## 可计数的语句类型

ExpressionStatement, VariableStatement, ReturnStatement, ThrowStatement, IfStatement, ForStatement, ForInStatement, ForOfStatement, WhileStatement, DoStatement, SwitchStatement, TryStatement, BreakStatement, ContinueStatement。

## 权重规则

每条语句的权重：

%% [语句权重] = 1 + <嵌套深度> + <分支加成> %%

- **嵌套深度**：语句被以下节点包裹的层数——IfStatement, ForStatement, ForInStatement, ForOfStatement, WhileStatement, DoStatement, SwitchStatement, TryStatement, CatchClause。每层 +1。
- **分支加成**：语句自身属于以下类型时 +0.5——IfStatement, ForStatement, ForInStatement, ForOfStatement, WhileStatement, DoStatement, CaseClause, CatchClause, ConditionalExpression。

%% [加权语句数] = round(所有语句权重之和 * 10) / 10 %%
