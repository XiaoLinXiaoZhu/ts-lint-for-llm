# 诊断

分析器产出的问题报告，共 4 种。

## 错误（影响退出码）

| 诊断 | 含义 |
|------|------|
| 【missing_capability】 | 函数缺少被调方的能力 |
| 【undeclared】 | 函数未声明能力 |
| 【unregistered】 | 调用了未注册的函数，无法验证能力 |

## 信息（不影响退出码）

| 诊断 | 含义 |
|------|------|
| 【implicit_capability】 | 函数通过【自动检测】获得了未显式声明的能力 |

退出码规则：存在任何错误级诊断 → 退出码 1；仅有 info 或无诊断 → 退出码 0。
