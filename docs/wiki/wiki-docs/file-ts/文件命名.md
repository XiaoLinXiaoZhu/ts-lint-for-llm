# 文件命名

文件名使用 kebab-case，【编译器】自动转换为代码标识符。

| 文件 | 标识符 |
|------|--------|
| `toggle-play.fts` | `togglePlay`（camelCase） |
| `player-state.type.fts` | `PlayerState`（PascalCase） |
| `_internal-helper.fts` | `internalHelper`（去 `_` 后转 camelCase，产物中保留 `_` 前缀含义见【可见性】） |
