# 能力声明

【fts文件】首行若为 `/** @capability ... */`，【编译器】将其原样保留到【编译产物】中。格式与标准 .ts 文件的【JSDoc声明】完全一致，无简写、无翻译。

```
┌─ fetch-user.fts
│ /** @capability IO Async Fallible */
│ (id: string): Promise<User | null> =>
│   fetch(`/api/user/${id}`).then(r => r.json())
└─
```

纯函数用空 @capability 声明：

```
┌─ add.fts
│ /** @capability */
│ (a: number, b: number): number => a + b
└─
```

一个 .fts = 一个函数 = 一个 `@capability` = 一个【函数得分】评分单元。
