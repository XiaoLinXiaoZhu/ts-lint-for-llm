# 外部声明文件

以 `.cap.ts` 为后缀的文件，用于为项目依赖的外部函数预声明【能力】。

## 文件格式

标准 TypeScript declare 语法 + 【JSDoc声明】：

```ts
/** @capability IO Async Fallible */
declare function externalApiCall(): any;

declare const client: {
  /** @capability IO Async Fallible */
  chat(...args: any[]): any;
};
```

支持三种声明形式：
1. `declare function` — 独立函数
2. `declare const` + 类型字面量中的方法/属性签名
3. `interface` 声明中的方法/属性签名

## 查找规则

从项目根目录（tsconfig.json 所在目录）递归扫描所有 `*.cap.ts` 文件。跳过 node_modules、dist、`.` 开头的目录。

## 匹配方式

按函数名匹配。当【调用解析】未能解析到项目内函数时，在外部声明表中按被调函数名查找。

外部声明文件中声明的函数不会产生 【unregistered】 诊断。
