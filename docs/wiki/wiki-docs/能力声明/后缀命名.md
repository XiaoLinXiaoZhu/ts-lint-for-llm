# 后缀命名

【能力声明】方式之一。函数名用 `_` 分隔，其中与【能力】名完全匹配的部分即为声明。

```ts
function fetchUser_IO_Async() {}           // 声明 IO + Async
function getConfig_IO_HandleFallible() {}  // 声明 IO + HandleFallible
function pureCalc() {}                     // 无匹配 → 不算后缀命名，走其他声明方式
```

8 个能力名均可用于后缀：IO, Impure, Fallible, Async, Mutable, HandleFallible, HandleAsync, HandleMutable。

优先级最高：若后缀命名匹配到至少一个能力，直接采用，不再检查 JSDoc。
